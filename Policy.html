# 本サービス プライバシーポリシー

CatHouse Productsが提供するサービス（以下、「本サービス」）は、Discord上で提供するサービスにおけるユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシー (以下「当ポリシー」) を定めます。

# 保存、収集する情報

# 当サービスは、以下の情報を一時的に収集、保存する場合があります。

  • サーバー内のメッセージやメンバー、チャンネル情報
  • その他ユーザーが提供する情報

# 情報の利用目的

# 本サービスは、以下の目的で収集した情報を使用します。

  • Discord上での不正行為や迷惑行為の防止のため
  • 機能の提供および改善、ユーザーサポートのため

# 情報の共有

当botは、以下の場合に情報を第三者と共有することがあります。

  • 法的要求に応じた場合
  • 当botの権利、財産、安全を保護する必要がある場合

# プライバシーポリシーの変更

当ポリシーの内容は、必要に応じて変更することがあります。変更後は、当サイト上で公開される最新版のプライバシーポリシーが適用されます。
