# 本サービス 利用規約

この利用規約（以下、「本規約」といいます。）は、CatHouse Products（以下、「当方」といいます。）が提供するサービス（以下、「本サービス」といいます。）の利用条件を定めるものです。本サービスを利用する際には、本規約に同意したものとみなされます。

ユーザーは、本サービスの利用にあたり、以下の行為をしてはなりません。

  • 法令または公序良俗に違反する行為

  • 当方、他のユーザーまたは第三者に対する詐欺行為または強迫行為
  • 本サービスの不具合等を使用した荒らし行為
  • その他、当方が不適切と判断する行為

# 第2条（免責事項）

当方は、本サービスがユーザーの特定の目的に適合すること、本サービスが中断やエラーなく利用できること、本サービスに不具合が生じないこと、及び本サービスに関して当方が提供する情報が正確であることについては、何ら保証するものではありません。
当方は、本サービスの利用によって生じた損害について、一切の責任を負いません。

# 第3条（利用規約の変更）

当方は、本規約を変更することができるものとします。当方が本規約を変更した場合、変更後の規約は当サイト上に掲示した時点より効力を生じるものとします。

# 第4条（開発者の行動）

当方は、本サービスについての規約違反を見つけた場合 無断で該当するサーバーに参加することが可能であり、場合によっては本サービスの使用を禁止することができるものとします。
当方は、本サービスに関連するサーバーに参加する際に、第1条に定める禁止事項を遵守するものとします。

# 第5条（個人情報）

当方は、ユーザーから取得した個人情報を、ユーザーの同意を得ることなく第三者に提供することはありません。ただし、次の場合はこの限りではありません。

  • 法令に基づく場合
