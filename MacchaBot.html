# 抹茶Bot

抹茶Botは、抹茶ラテ (opens new window)が作成しているBotです。
基本的なコマンドに加え、他のBotでは中々見ないようなコマンドが複数搭載されており、使っていて便利なBotです。

# ボットの招待リンク

https://discord.com/api/oauth2/authorize?client_id=1143127126212612177&permissions=8&scope=applications.commands%20bot (opens new window)

# 抹茶Bot Commands

TIP

抹茶Botのコマンドの接頭辞は「/」です。
例: /avatar

# Main Commands

avatar
アバターを取得します。

embed
埋め込みを送信します。

ping
ping値を表示します。

server
サーバー情報を表示します。

status
Botのステータスを表示します。

user
ユーザー情報を表示します。

webhook
webhookを使用してメッセージを送信します。

# Moderation Commands

addrole
ユーザーに特定のロールを付与します。

ban
特定のユーザーをBANします。

banword
禁止ワードを設定できます。

banwordlist
登録されている禁止ワードを確認できます。

banworddel
特定の禁止ワードを削除します。

kick
特定のユーザーを追放します。

newrole
新しいロールを作成します。

remrole
特定のユーザーからロールを削除します。

timeout
特定のユーザーをタイムアウトします。

unban
特定のユーザーのBANを解除します。

untimeout
特定のユーザーのタイムアウトを解除します。

# Server Commands

authc
認証ボタンを作成します。

emojicreate
絵文字を作成します。

# Memo Commands

memo write
メモに書き込みます。

memo overwrite
メモの内容を上書きします。

memo content
現在書き込まれていることを確認できます。

memo delete
メモから内容を削除します。
