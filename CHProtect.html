# CH Protect

CH Protectは、当チーム(CatHouse Products)が作成しているBotです。
スパム対処、禁止ワードの設定・対処など、「荒らしの対処」を主としたBotになっています。
こちらは現在開発中のため結構ガバガバです。

# ボットの招待リンク

https://discord.com/api/oauth2/authorize?client_id=1147722867782451230&permissions=8&scope=applications.commands%20bot (opens new window)

# CH Protect Commands

TIP

CH Protectのコマンドの接頭辞は「/」です。
例: /help

# Main Commands

guide
セットアップ方法を表示します。

help
コマンドの一覧を表示します。

ping
Botの応答テストに使用します。遅延値も表示します。

# Moderation Commands

ban
指定したメンバーをBANします。

kick
指定したユーザーを追放します。

timeout
指定したユーザーをタイムアウトします。

untimeout
指定したユーザーのタイムアウトを解除します。

clear
指定したメンバーのメッセージを一定数削除します。

# Banword Commands

banword add
禁止ワードを追加します。

banword remove
指定した禁止ワードを削除します。

banword list
登録されている禁止ワードの一覧を表示します。

banword setting
指定した禁止ワードのタイムアウト時間を変更します。

# Setting Commands

# スパム検知

特定の時間内に数回連続で同じ発言をしたユーザーにタイムアウト処罰を実行します。
spam enableで有効化、spam disableで無効化できます。

# Token検知

発言内にDiscord内で使用されるTokenが含まれていた場合削除します。
token enableで有効化、token disableで無効化できます。
